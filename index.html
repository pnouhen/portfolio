Projet personnel

Refonte d’un site vitrine pour une nutritionniste

Cliquez ici pour accéder au projet

Objectif du projet :

Moderniser un site vitrine existant en y intégrant des fonctionnalités dynamiques orientées utilisateur, tout en facilitant la gestion de contenu pour l’administratrice (la nutritionniste).

Évolutions prévues :

Gestion de la connexion utilisateur sécurisée pour permettre l’achat de recettes.
Mise en place d’un espace administrateur pour modifier les tarifs et gérer les recettes (ajout, modification, suppression).

Fonctionnalités principales :

L’application propose une navigation multi-pages (Accueil, Qui-suis-je, etc.) avec des données (recettes, avis, tarifs) stockées dans MongoDB. Les recettes sont filtrables par catégorie et disposent d’une page dédiée. Les utilisateurs peuvent lire et publier des avis.

Compétences mises en œuvre :

Architecture front-end réactive avec React et routing multi-pages. API REST avec Node.js/Express connectée à une base MongoDB. Filtres dynamiques et affichage conditionnel des données.

Technologies utilisées

React, Vite, Tailwind, Node.js, Express et MongoDB

Explorer les réserves naturelles de France

Cliquez ici pour accéder au projet

Objectif du projet :

Créer une carte interactive affichant les réserves naturelles françaises avec des marqueurs personnalisés (polygons). Le but est de permettre à l’utilisateur de filtrer facilement ces réserves via plusieurs critères pertinents.

Fonctionnalités principales :

Recherche de réserves naturelles par nom, affichage des distances entre points, filtres par département, région, zone circulaire cliquée ou surface calculée. Affichage précis des réserves via polygones sur une carte interactive.

Compétences mises en œuvre :

Manipulation avancée de Leaflet, calculs géographiques avec Turf.js, filtres dynamiques, intégration d’une API, UI optimisée avec Tailwind, formatage de saisie avec Cleave.js.

Technologies utilisées

Leaflet, Turf.js, Vite, Tailwind CSS, Cleave.js.
